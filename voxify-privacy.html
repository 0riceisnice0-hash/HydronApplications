Privacy Policy for Voxify

Last updated: May 20, 2026

Voxify is a Chrome extension that modifies microphone audio for browser calls, mic testing, soundboard playback, and experimental AI voice conversion.

Data Voxify Collects

  • A random anonymous install identifier used for settings, analytics, and unlock checks.
  • Anonymous usage analytics such as install events, selected voice effect, popup opens, session duration, soundboard play counts, and non-sensitive diagnostics.
  • Feedback submitted through the extension, including an optional email address only if the user provides one.
  • Discord account identifier and membership status when a user chooses to verify Discord membership to unlock experimental voices.

Audio Data

Standard voice effects and soundboard mixing are processed locally in the browser.

Experimental AI Plus voice conversion is different. When a user selects AI Plus, Voxify records short microphone phrases, sends those phrases over HTTPS to the Voxify AI engine for conversion, then returns the converted audio to the browser so it can be played into the microphone stream.

Voxify uses AI Plus audio only to provide the requested conversion. Voxify does not intentionally store user voice input or converted voice output.

Third-Party Services

  • Cloudflare hosts the Voxify backend, entitlement checks, and AI engine routing.
  • Discord OAuth is used only for optional Discord membership verification.
  • Analytics and feedback may be forwarded to a private Discord channel for the developer.

Permissions

  • storage: saves user preferences and extension settings.
  • alarms: manages anonymous usage-session timing.
  • Website host access: lets Voxify modify microphone streams on pages where the user chooses to use voice effects.
  • Voxify backend and AI engine host access: supports unlock checks and AI Plus conversion.

Voxify does not request the tabs permission.

Limited Use

Voxify uses user data only to provide and improve the extension's visible features. Voxify does not sell user data, use it for unrelated advertising, or transfer it for unrelated purposes.
