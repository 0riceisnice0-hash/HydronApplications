Privacy Policy for Voxify

Last updated: April 10, 2026

Voxify is a Chrome extension that modifies microphone audio in real-time to apply voice effects during browser-based calls and voice chats.

Data Collection

Voxify collects anonymous usage analytics to improve the extension. This includes:

  • Install and update events
  • Which voice effects are used and for how long
  • Session duration
  • A randomly generated anonymous identifier (UUID) created on install — this is not linked to any personal information

No names, email addresses, IP addresses, or other personally identifiable information are collected automatically.

Feedback submitted voluntarily through the built-in feedback form may include an optional email address if the user chooses to provide one for follow-up purposes.

Audio Data

All audio processing occurs entirely within your browser using the Web Audio API. No audio is ever recorded, stored, uploaded, or transmitted to any server. Voxify does not have access to the content of your conversations.

Third-Party Services

Anonymous analytics data and user-submitted feedback are sent to a private Discord webhook endpoint operated by the Voxify development team. No data is shared with, sold to, or transferred to any other third parties.

Permissions

  • storage: Saves your preferences (selected voice effect, volume, pitch, favourites, soundboard slots) so they persist across browser sessions.
  • tabs: Communicates your current voice effect settings from the extension popup to the content script running in the active tab.
  • Host permission (<all_urls>): Content scripts run on all websites because the extension needs to intercept microphone access on any site that uses it (Discord, Google Meet, Zoom, Teams, Roblox, etc.).
  • Host permission (discord.com/api): Used solely to send anonymous analytics and voluntary user feedback to a private Discord webhook.

Data Retention

Anonymous analytics data is retained indefinitely in a private Discord channel accessible only to the development team. Users can uninstall the extension at any time to stop all data collection.

Your Rights

You can disable analytics by uninstalling the extension. No account is required to use Voxify, and no personal data is collected that would need to be deleted.
